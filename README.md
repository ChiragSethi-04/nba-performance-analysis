# NBA Player Performance Analysis

Statistical regression analysis predicting NBA player net rating.

## Project Structure
- `notebooks/` — Jupyter notebooks for each analysis section
- `outputs/` — Generated plots and tables
- `report/` — Final report

## Setup
1. Clone this repo
2. Download dataset from: kaggle.com/datasets/justinas/nba-players-data
3. Place all_seasons.csv in the data/ folder
4. pip install pandas numpy matplotlib seaborn scikit-learn statsmodels
